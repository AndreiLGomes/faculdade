TAGS!

Tags Básicas

  • h1 ... h6 = usado para tags de cabeçalho, usado para diferentes nivels de importância
  • p = Define um parágrafo para o texto
  • a = define um hiperlink
  • ul e ol = definan listas não ordenadas e ordenadas
  • li = defina um item dentro do ul ou ol
  • div e span = Tags de divisão usadas para agrupar e estilizar o conteúdo
  • br = insere uma quebra de linha
  • hr = insere uma linha horizontal

Tags adicionadas/Semânticas

  1. article = para conteúdo autônomo e independente
  2. aside = para conteúdo relacionado ao conteúdo circulante
  3. details e summary = para cirar conteúdo expansível
  4. figcaption = para legenda de elementos
  5. figure = para encapsular conteúdo autocontido
  6. main = para conteúdo principal do documento
  7. mark = para destacar um texto
